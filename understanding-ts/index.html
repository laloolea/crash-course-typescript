TypeScript Notes

tsc --init for compiling whole project

tsc for compiling

tsc name of the app.ts for compiling specific

tsc -w or tsc --watch to compiling every saving

If you enable sourceMap in the tsconfig.json it will allow you to see the source of your ts for debugging

outDir you can set the dir where your js files will be located

rootDir all the files that source will be compiled otherwise no

noEmitOnError set it to true and it will not create a js file if you have an error in your ts file
